# NJGROUP Website

Official static website for NJGROUP circular rubber technology platform.

## Core Positioning

NJGROUP — Lastik Kalitesinde Rejenere Kauçuk ve Döngüsel Kauçuk Teknolojileri Platformu.

## New Core Pages

- `index.html` — homepage focused on tire-grade reclaimed rubber.
- `tire-grade-reclaimed-rubber.html` — Google Ads landing page for tire-grade reclaimed rubber.
- `co2-devulcanization-technology.html` — CO2 assisted devulcanization technology overview.
- `eu-standard-workflow.html` — EU-standard oriented workflow, layout, equipment, environmental control and QC.
- `rubber-factory-solution.html` — reclaimed rubber plant project solution.
- `government-esg.html` — government, municipality and ESG cooperation page.
